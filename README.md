# Fabric-Defect-Detection-Dataset

## Description
Data samples containing fabric defects of 7 different types. These types include baekra, stain, color issues, selvet,gray stitch, cut and contamination.

## Structure
- train
  - images
  - labels
- test
  - images
  - labels
- valid
  - images
  - labels
